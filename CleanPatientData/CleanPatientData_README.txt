CleanPatientData HIPIE Plugin

This plugin cleans patient data file. 

INCOMING DATA NOTES

* The plugin expects patient data file to be in ICD10 layout.

* Patient Key is required field.

* Decent population of first, last, address line1 , city, state and zip are required for cleaner to function properly.

Cleaned ATTRIBUTE NOTES

* The plugin has custom logic to clean patient file. 

* Additionally this cleaner appends cleaned name and address.
